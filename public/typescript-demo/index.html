To compile the typescript you can run one of two options

  • npm run-script build
  • npm run-script watch

Similar to how nodemon refeshes on js file change, the tsc watch option watches typescript files for changes and then compiles them on the fly. Try out both to get a feel for them.

As an asside, We don't necessarily need nodemon to start the server because I don't see us changing server.ts often enough to benefit from the change. Feel free to take this thought with a grain of salt.
